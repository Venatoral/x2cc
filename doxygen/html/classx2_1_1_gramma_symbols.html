Detailed Description

对于typeString而言 如果是TYPE_TERM,则长度为至少是1，一般情况下是1 如果是TYPE_EMPTY,可以是任意值 如果是其它的，长度至少是1，并且建议大写字母

TYPE分三种类型：终结符 空符号 变量 空符号需要做单独处理，但是不属于存储级别的单独处理

任何时刻，保证表中的empty有且只有一项 – 实现方式就是令将EMPTY添加到存储结构之后，缓存返回的EMPTY index EMPTY 只支持初始化，不支持修改

这个类将由其它所有的类共享 empty项必须存在，且仅有一项,且其下标必须是EMPTY_INDEX, EMPTY_INDEX,EMPTY_STRING 需要存储在容器之中; 不可以删除empty项 初始化symString时使用UNDEFINED_STRING

内置的属性值：TYPE_UNDEFINED "UNDEFINED"

仅支持添加操作，暂不支持删除操作

可以使用string来添加文法，查询文法符号
